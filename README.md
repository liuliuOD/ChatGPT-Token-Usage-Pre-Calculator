# ChatGPT Token Calculator

## This is a tool that pre-calculates the amount of tokens you will consume before using ChatGPT.

---

### Support models and encoding methods:

- models

  > - davinci
  > - text-davinci-002
  > - text-davinci-003
  > - text-embedding-ada-002

- encoding methods

  > - cl100k_base
  > - p50k_base
  > - r50k_base
  > - gpt2

---

### Run in terminal:

```bash
$ poetry install
$ poetry shell
$ python3 main.py
```
